# Sound Assets

## new-order.mp3

This directory should contain a `new-order.mp3` file for the kitchen screen notification sound.

### How to add a custom sound:

1. Find or create a pleasant notification sound (not jarring)
2. Convert to MP3 format
3. Save as `new-order.mp3` in this directory

### Fallback:

If no MP3 file is present, the kitchen screen will use a Web Audio API generated beep as a fallback.

### Recommended sound characteristics:

- Duration: 0.5-1.5 seconds
- Volume: Moderate (not startling)
- Tone: Pleasant, attention-grabbing but not annoying
- Format: MP3, 128kbps or higher

### Free sound resources:

- freesound.org
- zapsplat.com
- notification-sounds.com
